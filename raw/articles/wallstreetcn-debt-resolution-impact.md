# 化债为重，刺激随后，影响几何？

**来源**: 华尔街见闻 / 中金固收研究 | **发布日期**: 2024年11月9日
**分析师**: 许艳、耿安琪、范阳阳、陈健恒
**URL**: https://wallstreetcn.com/articles/3733535

## 核心结论

> 10万亿化债规模略超预期，但"金边属性"弱化趋势不变。2027年6月城投退平台完毕后，3年以上城投债信用分析逻辑从刚兑信仰→区域基本面。

## 10万亿化债结构

| 措施 | 规模 | 时间 |
|------|------|------|
| 一次性增加专项债务限额 | 6万亿 | 2024-2026（分三年） |
| 每年专项债安排化债 | 8000亿/年 | 2024-2028（五年） |
| **合计** | **10万亿** | |

效果：2028年前隐性债务从14.3万亿→2.3万亿

## 城投债务规模

| 口径 | 规模 |
|------|------|
| 1600家有公募债的发行人总债务余额 | ~45万亿 |
| 宽口径总债务余额 | ~61万亿 |
| 总负债余额 | ~68万亿 |

> 化债仅针对官方口径隐债，城投平台经营性债务规模更大

## 城投债净增量预测

- 2024年（1-10月）：净增-1316亿（去年同期1.19万亿）
- 2025年预测：净增量仍接近0（坚持"借新还旧"原则）

## 2027年6月关键节点

> "城投退平台"完毕后，剩余久期超过3年的城投债"金边属性"弱化
- 分析逻辑从刚兑信仰→区域和企业基本面
- 各区域再融资能力分化
- 普通地方国企信仰排序：城市运营服务 > 资本运营 > 产业国企

## 2025年政府债券供给预测

| 项目 | 预测 |
|------|------|
| 赤字率 | 4%（+1ppt） |
| 赤字规模 | ~5.5万亿 |
| 专项债新增限额 | 4.2万亿 |
| 超长期特别国债 | ~1万亿 |
| 化债限额 | 2万亿 |
| **政府债券净增量合计** | **13-13.7万亿** |

## 货币政策

- 参照2015年债务置换期间：降准累计-150bp，逆回购利率-160bp
- 预计2025：短端下行>50bp，长端下行20bp+
- 社融增速：8.1%→8.6%

## 核心判断

> "政府部门接替私人部门加杠杆，私人部门尚未看到明显加杠杆迹象"
