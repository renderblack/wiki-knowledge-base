# 财政部披露化债半程"成绩单"：1年隐债减少近4万亿

**来源**: 财联社 | **日期**: 2025年9月12日 | **记者**: 杨斌
**URL**: https://www.cls.cn/detail/2143802

## 核心数据

| 指标 | 数据 |
|------|------|
| 2024年末全口径政府债务 | **92.6万亿** |
| 政府负债率 | **68.7%** |
| 2024年末隐性债务 | **10.5万亿**（较2023年减少近4万亿） |
| 债务平均利息成本降幅 | **>2.5个百分点** |
| 节约利息支出 | **>4500亿**（截至8月） |
| 融资平台退出进度 | **超6成**（截至2025年6月） |

债务构成：
- 国债：34.6万亿
- 地方法定债务：47.5万亿
- 地方隐性债务：10.5万亿

## 隐债清零进展

- 广东、北京、上海 + 22个地级市 + 113个县区宣告"全域隐债清零"
- 内蒙古：第一个退出化债重点省份
- 7月：吉林成为第二个退出省份

## 融资平台退出进度

| 时间节点 | 目标 |
|---------|------|
| 2024年末 | 40%融资平台退出（vs 2023年3月） |
| 2025年6月 | 超6成融资平台退出 |
| 2025年末（预计） | 70-80%退出 |
| 2026年末（预计） | 超9成退出 |
| 2027年6月 | 融资平台全部退出 |

## "加减乘除"四要点

财政部提出"十五五"期间化债四原则：

| 要点 | 含义 |
|------|------|
| **减法** | 存量上做减法：提前下达2026年化债额度，靠前使用 |
| **加法** | 管理上做加法：强化专项债"借用管还"全周期管理，隐性债+法定债"双轨"合并管理 |
| **乘法** | 效益上做乘法：科学安排规模结构，提升债券资金使用绩效 |
| **除法** | 风险上做除法：强化监测预警，零容忍遏制新增隐债，终身问责+债务问题倒查 |

## 关键判断

- 2028年底化债目标有望实现
- 置换债6万亿中4万亿已发行（截至2025年8月）
- 化债打通资金链条，地方腾挪出更多资源用于经济发展
