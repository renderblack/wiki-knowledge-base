# 置换债

## Overview

中央政府授权地方政府发行的新债券，用于置换存量的高成本隐性债务。本轮化债的核心政策工具。

## Key Facts

| Fact | Data |
|------|------|
| 总规模 | **6万亿** |
| 实施周期 | 2024—2026年（每年2万亿） |
| 2026年进度（截至3月29日） | 已发行约9603亿，占全年额度**48%** |
| 2026年进度（截至4月17日） | 已发行约1.13万亿，占全年化债盘子约40%，整体发行进度>55% |
| 性质 | 专项债务限额，纳入政府性基金预算 |

## 2026年化债总盘子

| 来源 | 规模 |
|------|------|
| 再融资专项债券（置换债） | 2万亿 |
| 新增专项债（化债用途） | 8000亿 |
| **合计** | **2.8万亿** |

## 实施机制

```
地方政府发行置换债（低息）
    ↓
募集资金用于偿还存量隐性债务（高息）
    ↓
隐性债务 → 显性债务
    ↓
付息压力 ↓ 财政空间释放
```

| 甘肃、西藏、陕西西安/延安、广西来宾 | 年内清零目标 |

## 政策评价

- **减负效果**：付息压力大幅减轻，节约利息>4500亿/年
- **精准性**：债务压力大的省份获得重点额度倾斜
- **监管高压**：财政部对新增隐性债务"零容忍"，发现一起查处一起问责一起

## 局限性

- 仅是"借新还旧"，不减少总债务规模
- 中央信用背书，但未解决地方政府过度举债的体制根源
- [[hidden-debt|隐性债务]]仍通过城投平台持续产生
- 2万亿额度发完后，约4.5万亿剩余隐债需地方自行化解

## 后续节点

- **2026年末**：隐债预测降至2.5万亿
- **2027年6月**：化债首个重要观察时点；约9万亿城投债到期高峰，市场关注后续政策
