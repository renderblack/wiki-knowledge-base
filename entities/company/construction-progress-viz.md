# 兴土智瞰（施工进度可视化管理系统）

> **产品名**：兴土智瞰
> **项目编码**：G202501
> **版本**：V1.8，2024年9月立项，2025年11月18日交付（按时）

## 核心定位

施工进度「**看得见、风险早预警、数据可追溯、管理精细化**」的软硬件协同平台。

**解决痛点**：传统进度管理「不直观、数据散、风险隐、协同弱」
**产品口号**：计划编制 → 进度跟踪 → 资源调配 → 风险预警 → 报告生成

## 核心功能（8大模块）

| 模块 | 核心能力 |
|------|---------|
| 项目管理监测 | 基本信息、进度总览、实时看板 |
| 人员管理 | 劳务人员配置、出勤统计 |
| 设备管理 | 设备台账、使用统计 |
| 工程管理 | 桥梁结构、模型关联 |
| 任务分配 | 任务派发、进度跟踪 |
| 实时日报汇总 | 当天日报自动汇总 |
| 预警机制 | 滞后预警（强控型天数/量化型百分比） |
| 日报填写 | 微信小程序端便捷填报 |

## 关键概念

| 概念 | 说明 |
|------|------|
| 强控型任务 | 含前置任务+延隔时间，严格控制执行顺序 |
| 量化型任务 | 工时驱动，设定计划持续时间 |
| 关键路线 | 耗时最长且带前置任务设定的路线 |
| 进度偏差 | 自动计算，红滞后/蓝正常/绿提前 |
| 升降式展台 | 配套硬件，嵌入高清触控屏（86/98/110寸） |
| 装配式移动展房 | 快速搭建/拆卸展示空间（9×6/9×9/9×12m） |

## 技术栈

| 维度 | 技术选型 |
|------|---------|
| 开发语言 | Java（JDK 8 64位） |
| Web容器 | Apache Tomcat 8.5.69 |
| 数据库 | Microsoft SQL Server 2012 |
| 三维引擎 | 集成3D可视化引擎 |
| 权限模型 | RBAC（角色-权限-用户） |
| 客户端 | PC端 + 微信小程序 |
| 数据输入 | MS Project计划导入 + Excel人员/设备清册 |

## 硬件载体

软硬件协同：升降式展台（触控屏）+ 装配式移动展房，适配室内外展示场景。
