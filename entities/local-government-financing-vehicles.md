# 城投平台公司（LGFV）

## Overview

地方政府出资设立、承担政府投资项目融资功能的国有企业。是隐性债务的主要举债载体，2027年需完成全面退出。

## Key Facts

| Fact | Data |
|------|------|
| 隐性债务清零 | 江西抚州市42家平台隐债已清零 |
| 退出进度（2024年末） | ~40%（vs 2023年3月） |
| 退出进度（2025年6月） | **超6成** |
| 退出进度（2025年末，预计） | 70-80% |
| 退出进度（2026年末，预计） | **超9成** |
| 完全退出期限 | **2027年6月** |

## "退平台"背景

"退平台"= 城投公司不再承担政府融资功能，转型为市场化经营主体。
- 退平台后：隐债清零，债务从"政府债务"变为"企业债务"
- 未退平台：继续承担隐债偿还义务

## 城投债到期压力

| 时间节点 | 规模 | 性质 |
|---------|------|------|
| 2027年6月后到期/行权 | **约9万亿** | 占城投债总量超50% |

> 2027年6月是本轮化债后首个重要观察节点。届时城投债"金边属性"弱化，信用分析从刚兑信仰→区域基本面。

## "三资三化"方案

- **三资**：国有资本、国有资产、国有资源
- **三化**：资本化、资产化、资源化
- **目标**：盘活存量国有资产补充化债资金

## 关联分析

- LGFV 的持续举债是 [[hidden-debt|隐性债务]] 难以根治的体制根源
- 本轮化债通过 [[swap-bond|置换债]] 缓解短期压力
- 城投平台转型（[[debt-resolution|化债]]）是长期命题
- [[property-crisis|房地产]]下行削减了城投最依赖的土地出让金收入
