# 城投债到期压力

## Definition

城投平台公司在资本市场发行的城投债券（区别于隐性债务中的银行借款等）面临的大规模到期偿还压力。2027年6月后约9万亿集中到期，是本轮化债周期后最重要的观察节点。

## 关键数据

| 指标 | 数据 |
|------|------|
| 2027年6月后到期城投债 | **约9万亿**（占城投债总量超50%） |
| 城投平台有息债务（2024年） | **58.7万亿** |
| 其中短期/高息非标债务 | 约**17.8万亿** |
| 融资平台经营性债务（2025年末估算） | 约**7.26万亿**（财通证券） |

## 城投债 vs 隐性债务

| 类别 | 是否在化债范围内 | 规模 |
|------|----------------|------|
| 纳入隐债系统的城投债务 | ✅ 是 | ~10.5万亿（2024年末）→2.5万亿（2026年末预测） |
| 城投平台在资本市场发行的城投债券 | ❌ 否（直接） | 约9万亿（2027年6月后到期） |
| 城投平台全部债务（含经营债务） | ❌ 否 | 58.7万亿 |

## 时间轴分析

### 阶段一：化债周期（2024-2027年6月）

```
2024年11月  一揽子化债政策出台（10万亿）
     ↓
2024-2025  隐债置换大规模推进
     ↓
2025年7月  内蒙古第一个退出化债重点省份
     ↓
2025年    372家城投公司公开宣布"退平台"
     ↓
2026年     化债收官之年
          → 2万亿置换债发行（截至4月17日进度>55%）
          → 5地隐债清零（厦门、内蒙古、吉林、宁夏、新疆）
          → 34个城市公布隐债清零进展
          → 44个城市披露清欠进展
     ↓
2027年6月  化债首个重要节点
           → 隐债目标降至2.3万亿
           → 融资平台全部退出
           → 城投债"金边属性"弱化
           → 约9万亿城投债到期高峰
```

### 阶段二：城投债到期高峰（2027年6月后）

- 约9万亿城投债在2027年6月后到期或行权
- 城投平台退平台后，与政府的信用关联减弱
- 市场开始用纯企业信用分析框架评估城投债

## 2027年6月：关键转折点

### "金边属性"弱化

> 2027年6月"城投退平台"完毕后，剩余久期超过3年的城投债分析逻辑从**刚兑信仰**→**区域和企业基本面**

| 维度 | 化债前 | 化债后（2027.6后） |
|------|--------|------------------|
| 信用分析框架 | 地方政府隐性背书 | 区域财政实力 + 企业经营 |
| 再融资能力 | 有政府兜底预期 | 取决于区域经济活力 |
| 区域分化 | 不显著 | 显著分化 |

## 城投平台经营性债务（新战场）

> "多措并举化解地方政府融资平台**经营性债务**风险" — 中央经济工作会议

| 特点 | 数据 |
|------|------|
| 城投平台有息债务（2024） | **58.7万亿** |
| 其中短期/高息非标（潜在风险） | 约**17.8万亿** |
| 融资平台经营性债务（2025年末估算） | 约**7.26万亿**（财通证券） |

河南2026年预算：着力化解融资平台经营性债务风险，确保完成融资平台退出任务
江西2026年预算：推动剩余融资平台全部退出，促进改革转型

## 核心判断

1. **化债≠城投债无忧**：化债解决隐债，城投债9万亿到期不在化债直接覆盖范围
2. **2027年6月是压力测试点**：政府隐性背书退出，信用分析重构
3. **区域分化不可避免**：发达省市再融资能力强，弱资质省市面临再融资困难
4. **经营性债务是新战场**：7.26万亿+规模，下一个政策焦点
5. **净融资崩塌是预警信号**：市场已在提前收缩敞口
