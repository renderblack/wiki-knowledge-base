# 城投债到期压力

## Definition

城投平台公司在资本市场发行的城投债券（区别于隐性债务中的银行借款等）面临的大规模到期偿还压力。2027年6月后约9万亿集中到期，是本轮化债周期后最重要的观察节点。

## 关键数据

| 指标 | 数据 |
|------|------|
| 2027年6月后到期城投债 | **约9万亿**（占城投债总量超50%） |
| 1600家有公募债的发行人总债务 | ~45万亿 |
| 宽口径总债务余额 | ~61万亿 |
| 总负债余额 | ~68万亿 |

## 城投债 vs 隐性债务

> **关键区别**：化债针对的是纳入财政部监测系统的[[hidden-debt|隐性债务]]（银行借款+城投债中的隐债部分）。城投平台在资本市场发行的**城投债**（城投债券）不在化债政策直接覆盖范围内。

| 类别 | 是否在化债范围内 | 规模 |
|------|----------------|------|
| 纳入隐债系统的城投债务 | ✅ 是 | ~10.5万亿（2024年末） |
| 城投平台在资本市场发行的城投债券 | ❌ 否（直接） | 约9万亿（2027年6月后到期） |
| 城投平台全部债务（含经营债务） | ❌ 否 | 45-68万亿 |

## 时间轴分析

### 阶段一：化债周期（2024-2027年6月）

```
2024年11月  一揽子化债政策出台（10万亿）
     ↓
2024-2025  隐债置换大规模推进
     ↓
2025年7月  内蒙古第一个退出化债重点省份
     ↓
2025年6月  超6成融资平台退出
     ↓
2026年     化债收官之年，2万亿置换债发行
     ↓
2027年6月  化债首个重要节点
           → 隐债目标降至2.3万亿
           → 融资平台全部退出
           → 城投债"金边属性"弱化
```

### 阶段二：城投债到期高峰（2027年6月后）

- 约9万亿城投债在2027年6月后到期或行权
- 城投平台退平台后，与政府的信用关联减弱
- 市场开始用纯企业信用分析框架评估城投债

## 2027年6月：关键转折点

### "金边属性"弱化

> 2027年6月"城投退平台"完毕后，剩余久期超过3年的城投债分析逻辑从**刚兑信仰**→**区域和企业基本面**

| 维度 | 化债前 | 化债后（2027.6后） |
|------|--------|------------------|
| 信用分析框架 | 地方政府隐性背书 | 区域财政实力 + 企业经营 |
| 再融资能力 | 有政府兜底预期 | 取决于区域经济活力 |
| 区域分化 | 不显著 | 显著分化 |

### 区域利差扩大

中金固收研究判断：
- 弱资质省市：以化债为主，专项债额度向发达省市倾斜
- 城投债发行：坚持"借新还旧"，净增量预计仍接近0
- 区域再融资能力可能出现分化：城市运营服务类 > 资本运营类 > 产业类

## 城投债净融资崩塌数据

| 年份 | 城投债净融资 |
|------|------------|
| 2023 | 1.4万亿 |
| 2024（1-10月） | **-1316亿**（去年同期1.19万亿） |
| 2025（预测） | 接近0（坚持借新还旧） |

> 城投债净融资从1.4万亿→负值，意味着城投平台在资本市场的再融资能力已大幅萎缩。

## 城投平台经营性债务

中央经济工作会议首次明确：
> "多措并举化解地方政府融资平台**经营性债务**风险"

| 特点 | 说明 |
|------|------|
| 性质 | 城投平台退出融资功能后的"纯企业债务" |
| 规模 | 未公开，估算差异大 |
| 化解难度 | 比隐债更复杂，涉及市场化交易对手 |

胡恒松建议：
1. 精准摸排底数，将未列入融资平台名单的城投公司债务纳入监测
2. 推进基础设施领域 **REITs**，以稳定现金流资产偿债
3. 建立合规申报通道，将历史"漏报瞒报"债务纳入管理

## 核心判断

1. **化债≠城投债无忧**：化债解决的是隐债问题，城投债9万亿到期不在化债直接覆盖范围
2. **2027年6月是压力测试点**：城投平台退平台完成后，政府隐性背书退出，信用分析重构
3. **区域分化不可避免**：发达省市再融资能力强，弱资质省市可能面临再融资困难
4. **经营性债务是新战场**：隐债化解后，经营性债务将成为下一个政策焦点
5. **净融资崩塌是预警信号**：城投债净融资从正转负，说明市场已在提前收缩敞口
