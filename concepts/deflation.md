# 中国通缩

## Definition

物价总水平持续下降。反映需求不足、经济活力减弱。

## 关键数据

| 指标 | 数据 |
|------|------|
| 通缩持续时间 | **连续10个季度**（截至2025年底） |
| 2025年真实GDP | 比官方数字低约**11%** |
| 实际GDP增速 | 约**2.5-3.0%**（官方称5.2%） |

## 核心矛盾

> "没有任何经济体在持续通缩下实现5%实际增长"
> —— Rhodium Group

官方GDP与真实增长之间存在系统性偏差，主要在投资项：
- [[investment-collapse|固定资产投资]] 名义值同比 -11%
- 制造业投资增速从9.2%降至1.9%
- 房地产[[property-crisis|房地产危机]]导致土地财政崩塌

## 传导机制

```
房地产下行
  → 土地出让金锐减
  → 城投平台收入下降
  → 偿债能力恶化
  → [[hidden-debt|隐性债务]]风险上升
  → [[debt-resolution|化债]]需求迫切
  → [[fiscal-deficit|财政扩张]]
  → 货币宽松
  → 但信贷需求疲软（实际仅降息4bp）
  → 宽松效果有限，通缩持续
```

## 政策应对困境

- 货币宽松传导不畅：降息被"僵尸化"——借新还旧，不做新投资
- 财政扩张受制于[[fiscal-pressure|财政压力]]
- 结构性问题（房地产、人口、出口依赖）非短期政策可解
