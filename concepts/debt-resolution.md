# 化债

## Definition

通过债务置换、展期、降息、盘活资产等方式，降低地方政府的债务风险和偿债压力。本轮化债是中国财政史上力度最大的债务化解行动。

## 本轮化债背景

- 2021年后 [[property-crisis|房地产]]持续下行
- 土地出让金大幅减少 → 城投平台偿债能力恶化
- 隐性债务风险累积，中央被迫介入

## 政策工具箱（2026年更新）

| 工具 | 规模/说明 | 性质 |
|------|----------|------|
| 置换债 | 6万亿（2024-2026，每年2万亿） | 低息换高息，以时间换空间 |
| 新增专项债（化债） | 8000亿/年（2024-2028） | 补充政府性基金财力 |
| 2025Q4追加 | 5000亿（3000亿化债） | 清理拖欠账款、PPP |
| 2026年化债总盘子 | **2.8万亿**（置换债+专项债） | 化债收官之年全力推进 |
| 停息挂账 | 探讨中 | 极端方案，争议大 |
| 清欠专项债 | 50万以下合同/项目2025年底清零 | 清理政府拖欠企业账款 |

## 化债三阶段（推测）

1. **应急阶段**（2024-2026）：置换债大规模发行，遏制风险扩散，平台退出
2. **消化阶段**（2027-2028）：隐债降至2.3万亿，观察城投偿付能力（9万亿城投债到期高峰）
3. **体制重构**（2028+）：城投平台转型，财政纪律重建

## 2026年三大新战场

### 1. 隐性债务"清零冲刺"
- 34个城市公布隐债清零任务最新进展（2026年以来）
- 5地（厦门、内蒙古、吉林、宁夏、新疆）隐债清零已实现
- 44个城市披露偿还拖欠企业账款（清欠）进展
- 甘肃、西藏、陕西西安/延安提出年内清零目标

### 2. 经营性债务纳入化解
- 2026年政策重心：**从化解隐性债务→化解经营性债务**
- 城投平台有息债务总规模：**58.7万亿**（2024年）
- 其中短期借款、短期债券、高息非标债务：约**17.8万亿**（潜在风险）
- 融资平台经营性债务规模（2025年末估算）：约**7.26万亿**（财通证券）
- 河南、江西：将化解融资平台经营性债务风险纳入2026年财政预算

### 3. "退平台"加速
- 2025年372家城投公司公开宣布"退平台"
- 退平台=上半程，下半程是城投**纯市场化转型**

## 2026年预算：历史性集中化

> 中央政府将承担新增债务主力，**超过56%的新增债务工具**由中央背书。

| 指标 | 数值 |
|------|------|
| 全年新增债务合计 | **11.89万亿** |
| 广义赤字率 | **8.1%**（远超2023-2024年） |
| 官方赤字率 | 4%（对应5.89万亿赤字） |
| 新增专项债 | 4.4万亿 |
| 超长期特别国债 | 1.6万亿 |
| 准财政工具（政策性金融） | 8000亿（投向AI/低空经济/数字经济） |
| 财政金融协同基金 | 1000亿（刺激消费） |

### 三大战略转向
1. **从基建到消费**：以旧换新政策扩展至服务业
2. **准财政工具支持战略领域**：8000亿政策性金融工具投向AI、低空经济、数字经济
3. **统一大市场改革**：收紧地方补贴监管，遏制碎片化税收优惠

## 关键时间节点

```
2026年       化债收官之年
     ↓
2026年末     隐债预测降至2.5万亿
     ↓
2027年6月    化债首个重要时点
           → 隐债目标降至2.3万亿
           → 融资平台全部退出
           → 约9万亿城投债到期高峰
           → 城投债"金边属性"弱化
     ↓
2028年       隐债基本化解
           → 城投平台转型完成
           → 体制重构
```

## 争议与挑战

- **展期≠化债**：债务只是延期，并非减少
- **官方 vs IMF口径差异**：[[debt-gap|官方隐债10.5万亿 vs IMF估算60万亿]]
- **城投债二次压力**：2027年6月后9万亿城投债到期，后续政策不明
- **"虚假化债"风险**：部分地区可能通过虚构交易化假债
- **经营性债务规模不透明**：7.26万亿（估算）缺乏官方口径
- **城投债净融资崩塌**：2024年1-10月净融资-1316亿，市场提前收缩敞口
