# 房地产危机

## Definition

2021年以来中国房地产市场的深度调整——销量、房价、新开工全面下行，开发商大量违约。

## 关键数据

| 指标 | 变化 |
|------|------|
| 销售面积 | 持续下滑 |
| 新开工面积 | 较2021年峰值 **-75%** |
| 土地出让金 | 大幅减少 |
| 开发商违约 | 恒大、碧桂园等头部企业相继违约 |

## 连锁反应

```
房地产下行
  ↓
土地出让金锐减（地方财政主要来源）
  ↓
[[lgfv|城投平台]]收入下降
  ↓
城投偿债能力恶化
  ↓
[[hidden-debt|隐性债务]]风险爆发
  ↓
[[debt-resolution|化债]]政策出台
  ↓
[[investment-collapse|投资崩塌]]
  ↓
[[deflation|通缩]]
```

## 与化债的关联

- 房地产是本轮化债的**根本诱因**
- 城投平台大量业务与土地挂钩（土地整理、保障房等）
- 化债资金部分来自中央，但地方财政的根本修复依赖房地产软着陆
- 2026年稳地产政策持续出台，但效果有限

## 2026年展望

- 政府持续出台支持政策（收储、城中村改造等）
- 但居民购房预期未根本扭转
- 房地产软着陆仍是最大不确定性
