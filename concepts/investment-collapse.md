# 固定资产投资崩塌

## Definition

Fixed Asset Investment (FAI) 是中国经济核心引擎，2025年下半年出现超预期下滑。

## 关键数据

| 指标 | 2024年 | 2025年（截至11月） |
|------|--------|------------------|
| 整体FAI | 正增长 | 名义值同比 **-11%** |
| 制造业投资 | +9.2% | **+1.9%** |
| 房地产新开工 | — | 较2021年峰值 **-75%** |

## 驱动因素

1. **房地产下行**（核心）：[[property-crisis|房地产危机]]导致开发商停工、土地流拍
2. **城投退潮**：化债背景下，[[lgfv|城投平台]]不再大规模举债投资
3. **预期恶化**：企业看不清前景，停止新建产能
4. **出口透支**：2025年上半年抢出口，消耗了订单，下半年出口回落

## 信贷信号

| 指标 | 数据 |
|------|------|
| 整体人民币贷款增速 | 6.3% YoY（11月） |
| 居民经营贷款 | 4.1%（9年新低） |
| 居民借贷 | 1.1%（历史最低） |

→ 居民和企业都在去杠杆，没有人愿意借钱

## 经济含义

FAI 负增长意味着：
- 经济增长失去最大引擎
- [[deflation|通缩]]加剧
- 就业压力上升
- 2026年稳增长政策重点之一即"提振投资信心"
