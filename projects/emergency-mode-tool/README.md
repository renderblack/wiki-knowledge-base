# 应急模式识别工具

> 来源：「新时代佃农.txt」对话记录（2026.04）

## 概述

当你被情绪触发时，通过交互式问答快速识别正在启动的应急模式。

## 两个版本

| 版本 | 文件 | 特点 |
|------|------|------|
| Qwen3版 | `qwen-version.html` | 5题递进流程，题库式 |
| **Claude版（推荐）** | `claude-preferred.html` | 场景→身体→行为→内心→恐惧五维，多模式组合输出 |

用户明确偏好 Claude 版，作为主版本存档。

## 9种应急模式

1. **雷达模式** — 超警觉/盯信号
2. **交租模式** — 讨好/过度负责
3. **冻结模式** — 空白/说不出话
4. **断联模式** — 解离/感觉麻
5. **镇静剂模式** — 刷手机/无意识消遣
6. **替代满足模式** — 性/手淫/幻想止痛
7. **爆发模式** — 积累后的怒/吼/摔
8. **自责法庭模式** — 内化超我审判
9. **救火模式** — 立刻解决/控制局面

## 使用方式

直接在浏览器打开 HTML 文件即可使用，无需网络连接。
