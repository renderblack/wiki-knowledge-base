# 钢铁行业量化研究

## 概述

钢铁（螺纹钢）是基建和房地产的核心建材，其价格是经济活力的重要晴雨表。钢铁行业通过「钢材贸易商—建筑商—城投平台」的链条，与[[化债研究]]深度关联。

## 核心逻辑链

```
钢材需求 = 基建需求（城投主导）+ 地产需求
     ↓
钢材价格上涨 → 基建/地产成本上升
     ↓
城投平台施工成本增加 → 债务压力加大
     ↓
→ 影响 [[城投债]] 偿还能力
     ↓
→ 影响 [[化债研究]] 节奏和规模
```

## 核心指标体系

### 价格指标
| 指标 | 说明 | 来源 |
|------|------|------|
| 螺纹钢现货价 | 全国均价，日度 | 我的钢铁网 |
| 热轧卷板价格 | 制造业需求代理 | 我的钢铁网 |
| 铁矿石价格 | 成本端，进口依存度高 | 普氏指数 |
| 焦煤/焦炭价格 | 成本端，钢铁厂原料 | 煤炭市场网 |

### 供需指标
| 指标 | 说明 |
|------|------|
| 高炉开工率 | 供给侧，产能利用率 |
| 螺纹钢表观消费量 | 需求侧，周度频率 |
| 钢材社会库存 | 贸易商库存，反映囤货意愿 |
| 出口量 | 外需，近年高出口分流国内压力 |

### 情绪/预期指标
| 指标 | 说明 |
|------|------|
| 挖掘机开工小时数 | 工地活跃度，基建先行指标 |
| 水泥出货量 | 基建+地产同步指标 |
| 建筑业PMI | 月度，官方数据 |

## 量化研究框架

### 方向 A：价格预测
- 输入：铁矿石、焦煤、库存、开工率、宏观情绪
- 输出：螺纹钢周度价格预测
- 方法：时序模型（Prophet/LSTM）或 ML 回归

### 方向 B：需求拆分
- 拆解螺纹钢需求 = f(基建投资, 地产投资, 出口)
- 量化各分项贡献度变化

### 方向 C：钢铁—城投关联分析
- 钢材价格上涨 N% → 城投施工成本增加 X%
- → 量化对化债规模的影响
