# 差序格局

> 费孝通《乡土中国》核心概念。中国传统社会的底层操作系统。

## 核心概念

**定义**：以"己"为中心的同心圆波纹结构，每个人是圈子中心，波纹推及处产生联系。

**特征**：
- 自我中心主义（每个人都是圆心）
- 伸缩性（"穷在闹市无人问，富在深山有远亲"）
- 亲疏有别（对圈内人讲人情，对圈外人冷漠）
- 公私界限模糊

**vs 西方契约文化（团体格局）**：

| 维度 | 差序格局 | 契约文化 |
|------|----------|----------|
| 社会结构 | 水波纹/同心圆 | 捆柴/团体 |
| 基本单位 | 家/家族 | 个人 |
| 人际准则 | 特殊主义（看人下菜碟） | 普遍主义（一视同仁） |
| 信任机制 | 特殊信任（只信熟人） | 普遍信任（信制度和系统） |
| 冲突解决 | 调解/给面子 | 诉讼/按律执行 |

## 仪式化展演工具

| 工具 | 功能 |
|------|------|
| 酒桌文化 | 把生人快速"熟人化" |
| 泛家族称呼 | 语言上拉近距离（哥、姐、叔） |
| 随份子/送礼 | 人情债的记账与绑定 |
| 抢买单 | 争夺圈子中心的权力 |
| 排座次/C位 | 等级制度的物理外化 |
| 托关系/找中间人 | 跨圈子链接的信用担保 |

## 商业影响

**对传统行业（基建等）的具体表现**：
- 组织结构：核心层（嫡系）—中间层—外围层
- 领导风格：家长制，一言堂，恩威并施
- 用人机制：任人唯亲，熟人推荐，空降兵存活率低
- 制度执行：情大于法，公私界限模糊
- 商业合作：关系导向，喝酒应酬重于合同条款

**企业初创期（0→1）**：差序格局是助推器，低成本解决信任问题
**企业扩张期（1→100）**：差序格局成为绊脚石，阻碍人才专业化

## 未来趋势

1. **圈层收缩**："断亲"现象兴起，差序格局从"生存必需"退化为"情感奢侈品"
2. **契约打底**：商业规则重构，人情做"增量"而非"底座"
3. **数字信用**：算法/平台逐步替代"中间人"角色
4. **私域运营**：数字时代的新型差序格局
