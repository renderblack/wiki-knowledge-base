# 董事长2025年度报告 — 核心论点

> 来源：朱总2025年公司年会报告转录（近2万字）
> 时间：2025年初

---

## 一、外部环境：四个维度全面恶化

### 1. 质量安全标准↑（从严）

- **外观质量**：混凝土外观质量要求极高，模板面板要求全304不锈钢
- **安全防护**："你想死都死不了"——防护严密到连怎么掉都掉不下去、爬都爬不出去
- **移动互联网放大**：事故透明化，江苏12局拱桥倒塌、西南总部倒塌，全民皆知
- **结论**：行业高风险+透明化时代，只有一条路——本质安全

### 2. 商务环境恶化（从难）

- **合同审批**：三级管理层层把关，一个合同最长400天签不下来
- **计税支付**：审批流程极长，现金流周转效率大幅下降
- **支付条件恶化**：
  - 以前：先付定金/押金/租金再干活
  - 现在：无预付款、无押金，分期支付，且不执行
  - 形式：6个月~1年商业承兑汇票，甚至以房抵款
- **赖账打折**：欠1000万追三年，打折到950万结清，普遍现象

### 3. 竞争态势：传统优势消失

| 传统优势 | 当前状态 |
|---------|---------|
| 商业模式（设计施工+租赁） | 全行业通用，不再稀缺 |
| 产品创新 | 十几年后被模仿追平，同质化严重 |
| 规模效应 | 对手物资规模也很大，几公里变桥同台竞争 |

**竞争对手特征（压力来源）：**
- 规模更小，决策极快（老板现场拍板）
- 经营手段灵活（送礼、茅台、拍卖）
- 人力成本刚性问题：竞争对手29人交社保 vs 新图1432人
- **新图问题**：营销人员要层层汇报，决策链长，反应慢

### 4. 税务稽查强度历史最高

- 2025年是新图成立以来"最多频次、最强措施"的税务稽查
- 装备公司：电脑数据被拷走，查了三个月，最后只有3万问题
- 税务局：如果你认为3万OK，我就派人去车间一个环节一个环节查
- 客户食堂随手礼：三年30万礼品，计征受赠者个人所得税（不可操作）
- 国家税务总局方针：**"应收尽收"**——土地财政断流后，税务补位
- 预判：26年会更烦

## 二、四大风险分析

### 风险1：安全风险（猝死型）

**触发条件**：一座桥/一副挂篮出现质量/垮塌事故
- 像12局、大家庭级别的事故一旦发生 → 新图直接关门
- **本质安全三层次**：
  1. 设计三层审核（结构安全）
  2. 制造体系建设（材料/焊工/工艺/出厂）
  3. 售后服务标准化（培训客户减少使用风险）
- **现场三个关键岗位**：班长（不干活、只指挥）、司索工（真才实学）、起重设备操作工

### 风险2：资金风险（猝死型）

**数据**：
- 应收账款：6.6亿（经营收入8亿多，现金流入6.6亿）
- 银行贷款：平均18个月期限，每月还4000万
- **用不确定的钱（工程款不确定），去应对确定的事（银行到期还款）**

**五项应对**：
1. 严控融资规模（不要盲目扩张）
2. 调整期限结构（一年期→2-3年中长期，18个月→36个月）
3. 建蓄水池（水坝式经营，旱涝保收）
4. 严格全面预算制度
5. 严控应收账款账期规定

### 风险3：税务风险（消耗型）

- 合规反而被反复稽查——"你没问题是吧，那我去车间查"
- **应对**：
  1. 严格执行国家科技政策和税收法规
  2. 优惠政策项目建立完整证据链（研发费用加计扣除/高新企业优惠）
  3. 主动邀请税务局来指导，做好信息沟通

### 风险4：知识产权风险（侵蚀型）

- XT300自驾体系2024-2025年风靡全行业（长三角/石油/铁路100%覆盖）
- 竞争对手（原有A类厂家）业务受损，全力模仿
- 专利诉讼：部分专利被否定，知识产权律师团队不够强
- **应对**：建立更严密的知识产权体系 + 强大专利律师团队 + 钱要花下去

## 三、战略应对

### 1. 行为自律（风险控制）

- 安全是底线，三层本质安全体系
- 资金管理五措
- 税务合规+证据链
- 知识产权保护升级

### 2. 高效运营（降本增效）

#### 组织变革
- **原问题**：中心制（研发中心/营销中心/管理中心等）→ 各部门各自为政
- **新架构**：项目经理小组（技术+营销+施工+售后）→ 项目管理委员会（总经理+各中心总监）
- **目标**：打破部门墙，联合作战

#### 数字化路径
```
数据治理（基础）
  ↓
信息化（流程记录）
  ↓
数字化（结构化分析→决策辅助）
  ↓
AI智能化（AI自主判断+决策，人只监督）
```
- **痛点**：数据标准不统一——应收账款的定义，财务/营销/施工三个口径
- **教训**：U9与PMC系统上线失败——物资命名标准不统一，开会9次，刘总拍桌子
- **AI应用方向**：合同评审、方案设计、财务

#### 成本控制
- 提出"干毛巾拧出水"——实际上还有大量空间
- 各中心总监领队：提出可节省项目+指标+方法，总经办跟踪
- 节省越多，奖金包越大

### 3. 产品策略（四类产品）

| 产品 | 策略重点 |
|------|---------|
| 钢便桥 | 细分场景（保通桥/斜坡栈桥/海上桥/内河桥）；大项目领导亲自上；整合甲方资源 |
| 挂篮 | 发挥组合优势（租赁+施工）；淘汰落后产品加速新结构部署；保持45-50%市场份额 |
| 支架 | 知识产权保护；延长高毛利产品生命周期 |
| 主体施工 | 三个目的（人才培养高地+对外展示窗口+新产品试验）；控制成本+外观质量 |

### 4. 长期战略

**核心竞争力底层逻辑：性价比**
- 性价比 = 价值/成本
- 价值 = 满足客户需求的程度
- **极致性价比 = 价值最大化 + 成本最小化**

**两条腿战略**：
1. **高毛利产品（自驾研发）**：自主投资，投后租赁，追求高毛利高回报；前提：创新+知识产权保护
2. **普通产品（规模运营）**：追求规模+效率+低成本；通过五金仓、资产管理、翻新利用提升运营能力

**三大能力**：
- 创新能力（研发→价值）
- 风控能力（安全/资金/税务/IP）
- 运营能力（供应链/营销/项目管理）

## 四、激励与文化

### 利益分配原则
```
创造100万价值：
  客户 40%（让客户得实惠才有持续订单）
  员工 20%（工资+奖金+股权）
  国家 20%（税费，一分不少）
  股东 20%（剩余）
```

**实体企业的困境**：客户打折 + 员工要分 + 税费刚性的情况下，股东（固定资产投资者）承担最大风险

### 员工激励框架

1. **职责明确**：界定清楚才谈得上责任心
2. **意义注入**：个人职责→部门职责→中心职责→企业使命，让员工理解自己的工作价值
3. **评价机制**：及时表扬+教练式批评+公正心+责任心+理解包容
4. **扬长避短**：天生我材必有用，发挥优势而非补短板
5. **薪酬保障**：不低于社会平均水平，关键岗位争取以上

### 核心价值观

> **守正 · 创新 · 经济 · 动力**

- 守正：依法经营、诚信经营、尊重规律
- 创新：产品创新、知识产权保护
- 经济：极致性价比、让桥梁施工更经济
- 动力：利益分配（客户/员工/国家/企业）
