## "其他应收款"：万科的账本魔术

其他应收款是万科表外风险的核心载体：

| 指标 | 金额 |
|------|------|
| 2021年末其他应收款峰值 | 2660.43亿 |
| 2025年末 | 1753.96亿 |
| 流动资产占比 | 25%+ |
| 2025年前五名非关联方应收款 | 385.48亿 |
| 对应坏账准备 | 288.26亿 |
| 综合坏账率 | **88.95%** |

**关键问题**：前五名非关联方在短短1-2年账龄内形成如此高坏账，说明什么？
→ 资金往来本质是**借贷给空壳公司**，项目早已烂掉。

## 四家博商系空壳的应收款明细

| 公司 | 应收款 | 坏账率 | 特征 |
|------|--------|--------|------|
| 深圳源沣 | 99.77亿 | 72.50% | 最大单笔 |
| 汇成荣通 | 89.43亿 | 96.32% | 49%持股青岛万科紫郡 |
| 汇兴宏成 | 81.34亿 | 93.56% | 60%持股宁波万科东潮之滨 |
| 晨耀投资 | 53.53亿 | **100%** | 间接持有恒大1.6%，恒大已爆雷 |

**四家合计**：324.06亿应收，坏账288.26亿，整体坏账率88.95%

## 博商系套利的四类核心手法

1. **高价收购关联方资产**：形成167亿其他应收款（高于市场公允价值）
2. **鹏金所放贷**：向关联方发放年化10-24%贷款，赚取超额利差
3. **巨额坏账计提**：对关联方计提88.95%坏账，将亏损转移至表外
4. **烂账承接**：表外平台承接烂账，掩盖真实经营风险

## 鹏金所与博商系的资金闭环

```
社会投资者资金
    ↓ 购买鹏金所理财产品
鹏金所
    ↓ 股权收益权投资产品
博商系公司（汇成荣通/汇兴宏成/晨耀投资等）
    ↓ 资金进入万科项目
项目公司（青岛紫郡/宁波东潮之滨等）
    ↓
- 盈利：收益通过分红/利息回流博商系
- 亏损：形成坏账，转嫁万科报表
```
