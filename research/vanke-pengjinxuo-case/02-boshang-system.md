# 博商系 — 万科的"影子万科"

## 博商系核心成员

| 公司 | 定位 | 关键数据 |
|------|------|----------|
| 深圳博商顺泰实业有限公司 | 博商系持股平台，实控人为何卓 | 何卓曾任职万科资金部 |
| 博商资产管理有限公司 | 表外融资+烂账承接 | 2024年底亏损百亿级 |
| 深圳市汇成荣通实业有限公司 | 表外套取万科应收款 | 应收89.43亿，坏账96.32% |
| 深圳市汇兴宏成实业有限公司 | 同上 | 应收81.34亿，坏账93.56% |
| 深圳市晨耀投资咨询有限公司 | 持有恒大地产1.6024%股权 | 对恒大应收53.53亿，100%坏账 |
| 深圳市源沣实业有限公司 | 应收款方，最大单笔 | 应收99.77亿，坏账72.50% |

## 运作模式

```
万科品牌背书 + 小股操盘 + 高杠杆运作

1. 博商系公司作为万科项目的小股东（表面）
2. 万科通过"其他应收款"向博商系输送资金
3. 博商系用资金撬动更大规模项目
4. 项目收益内部化，亏损则转为万科坏账
5. 通过鹏金所发行理财产品，社会资金导入
```

## 关键数字

- 四家空壳公司累计应收：**324.06亿元**
- 综合坏账率：**88.95%**（坏账288.26亿）
- 单家最高：深圳源沣 99.77亿应收，坏账72.50%
- 这四家公司特征：注册资本1000万，参保人数为零，典型空壳

## 博商系 × 恒大的连接

晨耀投资通过深圳麒翔持有恒大地产1.6024%股权：
- 2017年：深圳麒翔以战投身份向恒大地产增资50亿
- 背后实控人：吉林地产商王占江
- 恒大爆雷→晨耀投资应收53.53亿，100%坏账

**这意味着万科通过博商系间接投资了恒大。**
