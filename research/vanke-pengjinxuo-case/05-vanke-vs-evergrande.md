# 万科 vs 恒大 — 双样本对比分析

## 核心差异

| 维度 | 恒大 | 万科 |
|------|------|------|
| **治理模式** | 许家印家族**绝对控制** | 混合所有制，**无实控人**，管理层操盘 |
| **崩塌触发点** | 2021年现金流断裂（三条红线+销售崩盘） | 2024年鹏金所逾期（表外风险暴露） |
| **国资关系** | 无国资股东，靠政府协调救助 | 深铁控股，持续输血335亿+ |
| **核心问题** | 高杠杆扩张+财务造假+分红掏空 | 表外融资+关联交易非关联化+国资持续输血 |
| **造假/违规** | 虚增收入/利润（审计失守） | 关联交易非关联化、信披违规 |
| **帮凶生态** | 普华永道（审计）、评级机构、摩根/中金（融资） | 鹏金所（平台）、博商系（影子公司）、审计机构（待查） |
| **员工伤害方式** | 恒大财富全员摊派+强迫购买理财产品 | 鹏金所跟投制，员工自愿加杠杆参与 |
| **处置结局** | 许家印被拘，企业资不抵债 | 国资接管，管理层刑拘，但仍有持续经营可能 |
| **债务窟窿** | 2万亿+ | 885亿（2025年单年亏损） |

## 共性规律：制度套利的不同路径

两种模式本质相同：**利用中国房企特有的制度环境实现"风险社会化、收益私有化"**。

| 共性 | 恒大表现 | 万科表现 |
|------|----------|----------|
| **金融平台募资** | 恒大财富（员工+社会） | 鹏金所（员工+社会） |
| **表外负债** | 涉及银行、信托、表外项目公司 | 博商系影子公司 |
| **高杠杆** | 三条红线前后的高负债扩张 | 表外高杠杆运营 |
| **关联交易转移** | 许家印家族分红+离婚+离岸信托 | 博商系高价收购+坏账转移 |
| **国资背书** | 战投（深圳控股等）退出后的政策协调 | 深铁直接控股+持续输血 |
| **审计失守** | 普华永道14年标准无保留意见 | 待查 |

## 对"混合所有制"改革的核心拷问

万科案例暴露的深层问题：

1. **"无实控人"是双刃剑**：股权分散避免了单一强人控制，但也没有人能有效制衡管理层
2. **国资参股≠有效监管**：深铁持股15.31%，持续输血335亿，但从未有效约束管理层的表外操作
3. **"民营化运作"成为挡箭牌**：万科管理层以"民企机制灵活"为由规避国资合规要求
4. **"无评估、无约束"的资金投放**：深铁335亿借款无规范资产评估，无市场化风险定价
5. **利益共享的缺失**：高管薪酬与业绩/股东回报完全脱钩

## 万科 vs 恒大的"员工伤害"对比

| | 恒大 | 万科 |
|--|------|------|
| **方式** | 恒大财富强制摊派，员工自掏腰包购买理财产品 | 鹏金所跟投制，员工自愿贷款加杠杆跟投项目 |
| **机制** | 全员营销+理财产品强制任务 | 跟投制度激励，高回报承诺诱惑 |
| **损失** | 恒大财富投资者本金无法兑付 | 跟投本金损失，鹏金所贷款成为坏账 |
| **规模** | 员工+社会投资者数百亿 | 鹏金所8亿+逾期，社会投资者受损 |

## 监管启示

1. **表外融资平台需要穿透式监管**：鹏金所/恒大财富都绕过了地产融资监管
2. **员工跟投制度需要专项检查**：激励机制变成风险转嫁工具
3. **混合所有制企业的国资监管真空**：现有监管体系对"国资参股+民企运作"缺乏适配工具
4. **关联交易非关联化是核心漏洞**：影子公司模式需要专项治理
