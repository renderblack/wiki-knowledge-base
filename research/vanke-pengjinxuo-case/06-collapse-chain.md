# 崩塌传导链 — 八阶段模型

> 来源：Bolt老大梳理，直接可对标使用的结构化分析框架

```
阶段一：制度层（红利期）
  行业增长红利 + 混合所有制治理优势
  └─ 万科成为"行业三好学生"，混合所有制改革标杆

阶段二：构建期（工具搭建）
  事业合伙人 + 跟投制 + 鹏金所搭建
  └─ 制度工具异化：跟投本应是激励，变成风险转嫁通道

阶段三：扩张期（集体行为失控）
  管理层集权 + 表外扩张 + 强制跟投
  └─ 博商系影子公司网络成型，鹏金所资金池扩大

阶段四：套利期（非法行为）
  资金挪用 + 高利转贷 + 利益输送
  └─ 博商系高价收购资产+鹏金所放贷10-24%年化+高管薪酬与业绩脱钩

阶段五：失守期（监管真空）
  内控空白 + 股东监督缺位
  └─ 深铁持续输血335亿，无评估无约束；深交所/证监会未穿透表外

阶段六：爆发期（信任崩塌）
  鹏金所爆雷 → 表外风险暴露 → 市场信任崩塌
  └─ 2024年7月，8亿逾期，市场传闻鹏金所超50亿

阶段七：受灾区（损失传导）
  员工/供应商/投资者受损 → 公司现金流承压
  └─ 跟投员工本金损失，供应商商票无法兑现，投资者理财无法兑付

阶段八：追责期（司法介入）
  高管集体涉案 → 2025起集中被查 → 2026司法追责
  └─ 祝九胜2025年10月被刑事拘留，何卓等多人被控制
```

## 与恒大崩塌链的对照

| 阶段 | 万科 | 恒大 |
|------|------|------|
| 阶段一 | 混合所有制+行业红利 | 城镇化红利+金融宽松 |
| 阶段二 | 跟投制+鹏金所 | 高杠杆+多元化扩张 |
| 阶段三 | 表外扩张+管理层集权 | 表内高负债+许家印家族控制 |
| 阶段四 | 资金挪用+高利转贷+利益输送 | 财务造假+分红转移+离岸信托 |
| 阶段五 | 内控空白+股东监督缺位（深铁无约束） | 审计失守+监管缺位 |
| 阶段六 | 鹏金所爆雷（2024.07） | 现金流断裂（2021） |
| 阶段七 | 员工/投资者/国资受损 | 业主/投资者/银行受损 |
| 阶段八 | 高管刑拘（2025-2026） | 许家印被拘（2023） |

**共性根因**：制度套利空间 + 监管真空 + 激励机制扭曲
