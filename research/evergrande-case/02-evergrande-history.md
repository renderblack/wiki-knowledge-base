# 02 — EVERGRANDE 恒大集团大事年表

## 企业档案

| 项目 | 内容 |
|------|------|
| 成立 | 1996年，广州 |
| 上市 | 2009年11月，香港联交所 |
| 主营业务 | 房地产开发为主，扩展至物业、金融、汽车、文旅等 |
| 审计师 | 普华永道（2009—2023.1） |
| 爆雷 | 2021年9月（恒大财富） |
| 破产清算 | 2023年 |

---

## 关键时间线

### 创立期（1996—2009）

| 时间 | 事件 |
|------|------|
| 1996 | 许家印在广州创立恒大地产，第一个项目"金碧花园" |
| 1997 | 亚洲金融风暴期间，逆势拿地扩张 |
| 2004 | 恒大从广州走向全国，开启全国扩张 |
| 2009 | 港交所上市，当年销售额307亿 |

### 高速扩张期（2010—2016）

| 时间 | 事件 |
|------|------|
| 2010 | 收购广州足球俱乐部，进入足球产业 |
| 2013 | 推出恒大冰泉，进军矿泉水行业 |
| 2014 | 投资900亿进入光伏产业 |
| 2016 | 棚改货币化+激进杠杆，成为中国第一大房企 |

### 登顶期（2017）

| 时间 | 事件 |
|------|------|
| 2017 | 销售额5010亿，登顶中国房企第一 |
| 2017 | 许家印以2900亿身家成为中国首富 |
| 2017.10 | 十九大报告首次提出"房住不炒" |

### 危机潜伏期（2018—2020）

| 时间 | 事件 |
|------|------|
| 2018 | 恒大金服改名"恒大财富"，继续向员工摊派理财销售 |
| 2020.8 | **三道红线出台**，恒大三线全踩 |
| 2020.11.8 | **借壳深深房回A失败**，引爆第一次资金链危机 |
| 2020.11 | **恒大商票首次逾期** |

### 爆雷期（2021）

| 时间 | 事件 |
|------|------|
| 2021.9 | **恒大财富爆雷**，400亿理财无法兑付 |
| 2021.12.3 | 恒大公告"无法履行担保责任"，广东省政府派驻工作组 |

### 崩塌期（2022—2023）

| 时间 | 事件 |
|------|------|
| 2022 | 销售额从2000亿断崖式下跌至130亿 |
| 2023.1 | 普华永道辞职 |
| 2023.8 | **许家印被曝"技术性离婚"** |
| 2023.9.28 | **许家印被依法采取强制措施** |

### 审判期（2026）

| 时间 | 事件 |
|------|------|
| 2026.4.13-14 | 一审开庭，8项罪名，当庭认罪 |
