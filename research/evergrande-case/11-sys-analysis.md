# 11 — SYS ANALYSIS 制度性原因分析

## 核心问题

为什么恒大能从一个小开发商膨胀到2.4万亿负债、爆雷后主犯还能转移资产出逃？

不是一个人、一个环节的问题，而是**整个系统都存在漏洞**。

---

## 制度漏洞地图

```
┌─────────────────────────────────────────────────┐
│              制度层面（供给侧）                    │

│  土地财政 ──→ 地方政府鼓励开发商拿地                │
│  金融自由化 ─→ 银行大量放贷给房地产                  │
│  三道红线出台太晚（2020）──→ 杠杆已积累20年          │
└─────────────────────────────────────────────────┘
                        ↓

│              市场层面（中端）                       │
```

## 预售制度的双刃剑

### 预售制的设计

中国商品房预售制（卖楼花）从香港学来：
- 购房者支付首付（通常30%）
- 开发商在房子竣工前就能拿到全款
- 这笔钱没有硬性监管要求必须用于本项目

### 预售制被滥用的路径

```
开发商拿到预售款
    ↓
挪用到其他项目/拿新地/分红
    ↓
原项目资金不足 → 烂尾
    ↓
购房者付了钱，但房子收不到
```

恒大的162万套烂尾楼，相当一部分就是这样来的。

### 反思

预售制不是恒大事件的原因，但缺乏预售资金监管的预售制，给了恒大挪用资金的制度空间。

## 中介机构的利益冲突

### 商业模式内生缺陷

| 中介类型 | 商业模式 | 内生缺陷 |
|---------|---------|---------|
| 审计机构（普华永道） | 客户付费 | 不敢得罪大客户 |
| 评级机构（穆迪/标普/惠誉） | 发行方付费 | 高评级=更多业务 |
| 投行/承销商 | 承销费 | 帮发行人卖债就有收入 |

**三方利益高度一致**：帮恒大发债、给恒大背书 → 收入增长
**三方都没有动力**：揭示风险、给低评级、拒绝承销

### 为什么恒大能14年无刹车

1. 普华永道收入依赖恒大
2. 评级机构收入依赖恒大
3. 银行依赖恒大带来的贷款业务
4. 地方政府依赖恒大带来的土地财政
5. **所有人都选择了沉默**

## 许家印个人的赌性

许家印的行为逻辑不复杂：

1. **用别人的钱赌**：贷款、预售款、理财资金、战略投资
2. **赌赢了是自己的**：分红、信托、离岸架构
3. **赌输了是别人的**：烂尾楼留给业主，债务留给银行

这是一种极端的**道德风险**——收益私有化、风险社会化。

---

## 为什么政府出手这么晚

三道红线是2020年8月出台的，但：
- 恒大的高杠杆模式从2016年棚改后就开始加速
- 2017年许家印已经成为首富，恒大已经是第一大房企
- **政策等了4年才出，给了恒大充足的"跑路"时间**

### 可能的原因

1. **稳增长压力**：房地产是经济支柱，不敢过早刺破
2. **路径依赖**：房地产带动GDP、带动财政收入、带动就业
3. **协调成本**：跨部门监管协调需要时间
4. **侥幸心理**：可能认为恒大能软着陆

## 历史的轮回

恒大不是第一个"大到不能倒"最终倒下的：

| 企业 | 年份 | 核心问题 |
|------|------|---------|
| 海南发展银行 | 1998 | 房地产贷款坏账 |
| 德隆系 | 2004 | 民营金融集团崩盘 |
| 明天系 | 2020 | 影子银行+跨境并购 |
| 海航 | 2021 | 激进并购+资金链断裂 |
| 恒大 | 2021 | 高杠杆+预售款挪用+造假 |
| 碧桂园 | 2023 | 高杠杆+销售下滑 |

每一次危机都有相似的剧本，但监管总是慢半拍。

---

## 对当前化债的启示

### 1. 城投平台的明股实债

城投平台的PPP项目、产业基金中大量存在明股实债。这部分隐性债务的规模至今难以摸清。

### 2. 预售资金监管

2021年后虽然加强了预售资金监管，但历史遗留问题仍在。"保交楼"政策是对预售制度漏洞的被动补救。

### 3. 中介机构的独立性

恒大之后，监管机构对审计机构和评级机构的监管明显加强。但商业模式的内生缺陷没有根本改变。

### 4. 跨境资本流动

中国企业家的离岸信托和跨境资产布局，在资本管制收紧的背景下，成为监管的盲区。

---

## 总结

恒大事件的本质是：**在一个存在系统性漏洞的制度环境中，一个极度激进的企业家，利用所有可用的制度空间，将风险转嫁给社会，将收益留给自己。**

这不是孤立的犯罪，而是一面镜子，照出了中国房地产金融化过程中的所有制度缺陷。
