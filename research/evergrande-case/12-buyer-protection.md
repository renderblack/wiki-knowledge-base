# 12 — BUYER PROTECTION 烂尾楼与购房者困境

## 核心矛盾

烂尾楼受害者的困境：

```
购房者：
1. 支付首付（通常30%—40%房款）
2. 向银行申请贷款，银行一次性把贷款打给开发商
3. 房子烂尾，无法交付
4. 但银行房贷还要继续还，否则上征信
5. 不还款 → 银行查封房产 + 冻结其他资产 + 起诉
```

**钱已经给了开发商，但房子没拿到，为什么还要还贷款？**

---

## 为什么购房者无法停止还贷

### 法律链条

1. 购房者与开发商：签署《商品房买卖合同》，支付首付款，约定交房时间
2. 购房者与银行：签署《个人住房贷款合同》，银行一次性支付全部贷款给开发商，购房者按月还款
3. 开发商与银行：开发商提供阶段性担保（房屋抵押前的担保）

**关键**：《贷款合同》是购房者与银行之间的独立合同，不因房屋是否交付而解除。

### 银行的操作逻辑

银行说：
- "你借的是贷款，不是房款"
- "房子烂尾是开发商的民事违约"
- "你的还款义务是独立的合同义务"
- "不还钱我就查封你的其他财产、上征信"

### 法院的判决

2022年，最高法有一系列判例：
- 部分法院判决：**购房人有权停止还贷**（因为购房目的无法实现，贷款合同应予解除）
- 但银行普遍上诉
- 现实是：**多数购房者不敢停贷**，因为一旦上征信，影响子女入学、出行等

---

## "进了赌场，出来背上了贷款"

这个比喻非常精准，背后的逻辑是一样的：

| 赌场类比 | 烂尾楼现实 |
|---------|---------|
| 赌徒进场，以为能赢钱 | 购房者付首付，以为能拿到房 |
| 赌场老板拿了钱跑路 | 开发商拿了钱，楼盘烂尾 |
| 赌徒出来背负债务 | 购房者在没拿到房的情况下继续还贷 |

本质都是：**拿着你的钱，干完自己的事，然后让你背债。**

## 预售制度的结构性缺陷

### 资金流向

```
购房者支付首付 + 银行贷款
    ↓
进入开发商账户（或者监管账户）
    ↓
开发商自由支配（拿新地、分红、还债）
    ↓
原项目资金不足 → 烂尾
    ↓
开发商：有限责任 → 申请破产 → 债务清零
    ↓
购房者：无限责任 → 继续还贷
```

### 为什么说这是结构性缺陷

正常的交易逻辑应该是：
- 我付钱，你交货
- 货不对板，我可以要求退款
- 你跑路，我有追索权

**但预售制把这个逻辑打破了：**
- 房子还没建，我就把钱给了开发商
- 开发商拿了钱，建不建完我说了不算
- 建不完，我的钱打了水漂，但贷款还要还

## 恒大的具体数字

| 项目 | 数字 |
|------|------|
| 烂尾楼数量 | **162万套** |
| 受影响业主 | **600万人** |
| 涉及省份 | 全国31省 |
| 恒大未竣工项目 | 数百个 |

这162万套不是冷冰冰的数字，是162万个家庭——可能是一个家庭六个钱包凑出来的首付，每个月省吃俭用还月供，结果房子说烂尾就烂尾了。

---

## 保交楼政策

2022年以来，政府推出"保交楼"政策：

| 措施 | 内容 |
|------|------|
| 专项借款 | 国家借给开发商资金用于完工 |
| 地方国企接盘 | 地方政府安排国企接手烂尾项目 |
| 预售资金监管加强 | 防止预售款再被挪用 |
| 债务重组 | 允许开发商展期还债，以时间换空间 |

**但现实**：保交楼进展缓慢，很多项目资金缺口巨大。

## 社会影响的几个维度

### 1. 家庭层面

- 六个钱包凑首付 → 打水漂
- 月收入的50%+还月供 → 生活拮据
- 房子烂尾 → 婚姻矛盾、家庭破裂
- 心理创伤 → 失去对社会的信任

### 2. 社会稳定层面

大量烂尾楼业主维权：
- 断供潮
- 集体上访
- 与地方政府的冲突

### 3. 金融系统层面

大量购房者在"要不要断供"之间犹豫：
- 断供 → 银行坏账增加
- 不断供 → 购房者持续失血，消费萎缩

---

## 这件事的制度意义

烂尾楼问题的本质是：**中国购房者承担了不应该由他们承担的风险。**

正常市场经济的逻辑：
- 预售制应该有硬性资金监管：预售款只能用于本项目
- 应该有开发商履约担保：烂尾时购房者有追索
- 应该有购房者保护机制：开发商违约时贷款合同应可解除

**恒大的案例证明：这三条机制全部缺失。**
