# 13 — INDUSTRIAL CHAIN 产业链冲击

## 房地产产业链

恒大不是一个孤岛，它背后是整条产业链：

```
上游：
├── 钢铁/水泥/玻璃
├── 建材供应商（瓷砖、涂料、管材）
├── 工程机械（塔吊、挖掘机）
└── 设计院/勘察院

中游：
├── 建筑总包
├── 分包商（水电、装修、消防）
└── 园林/景观

下游：
├── 装修公司
├── 家具家电
├── 物业服务
└── 中介代理
```

恒大倒了，这条链上的每一个环节都在承受损失。

| 应付账款（各类供应商） | 数千亿 |

## 商票：最先倒下的多米诺骨牌

### 什么是商票

商业票据（商票）：开发商开具的延期付款凭证，承诺在未来某个日期支付货款。

供应商接受商票的原因：
- 恒大是大客户，不想失去订单
- 商票有恒大信用背书，以为能兑现
- 持票可以转手流通

### 恒大商票的规模

恒大商票在2020年11月首次出现逾期，随后大规模拒付。

供应商持有的商票无法兑现：
- 要么持有至到期（不知道什么时候）
- 要么折价转让（通常5-7折）
- 要么起诉追偿（赢了也没钱执行）

### 商票逾期的连锁反应

```
恒大商票逾期
    ↓
供应商收不到款
    ↓
供应商无法支付自己的上游
    ↓
整个链条出现债务三角债
    ↓
装修公司：没有现金流，无法采购材料、支付工人工资
    ↓
装修公司倒闭
```

---

## 装修公司的案例

装修公司是恒大倒下的"重灾区"之一：

### 典型路径

1. 恒大与装修公司签约：为恒大项目提供精装修服务
2. 恒大支付部分预付款，但大部分款项以商票形式结算
3. 装修公司：购买材料、雇工人、按时完工
4. 恒大商票到期 → 拒付或无力支付
5. 装修公司：材料商来催款、工人来讨薪
6. 装修公司：资金链断裂 → 倒闭

### 规模估算

恒大在全国有大量精装修交房项目，涉及数千家装修公司。

恒大倒下后，大量装修公司：
- 资金链断裂
- 员工失业
- 供应商货款无法追回

## 建材供应商

### 案例：某瓷砖供应商

- 恒大是某瓷砖厂商的最大客户之一
- 年供货量数千万

- 商票违约后，厂商资金链紧张
- 被迫削减产能、裁员

### 钢铁行业

恒大使用的钢筋量是天文数字。

恒大倒下后：
- 钢铁厂订单减少
- 钢铁厂应收账款无法回收
- 钢铁厂被迫压缩产能

## 建筑分包商

### 农民工工资问题

建筑行业大量使用农民工，层层分包：

```
恒大（甲方）
    ↓ 总包（建筑公司）
    ↓ 分包（劳务公司）
    ↓ 包工头
    ↓ 农民工
```

恒大欠总包 → 总包欠分包 → 分包欠包工头 → 包工头欠农民工

**农民工是整条链上最脆弱的一环**：
- 干活的时候没签正规合同
- 拿不到钱也不知道找谁
- 没有社保、没有法律保护

## 恒大商票的特殊性

恒大商票有几个特征：

1. **流通性强**：恒大商票在市场上广泛流通，大量供应商持有
2. **规模大**：恒大是房地产行业最大的商票出票人之一
3. **逾期公开**：2020年恒大商票拒付，信息公开后引发市场恐慌

**恒大商票的违约，是恒大债务危机第一个公开化的信号。**

---

## 三角债：无解的困局

恒大倒下后，整个房地产产业链陷入"三角债"困局：

```
甲方（恒大）欠乙方（总包）
乙方欠丙方（分包）
丙方欠丁方（材料商）
丁方欠戊方（工人工资）
戊方没有钱
```

要解开这个结，需要：
- 要么甲方有钱（恒大没有）
- 要么政府介入（已经在推动）
- 要么各方让步（谁都不愿意）

## 对经济的影响

### GDP层面

房地产产业链对GDP的贡献超过25%。

恒大倒下：
- 直接减少的房地产开发投资
- 间接减少的上游建材需求
- 连锁导致的消费萎缩

### 就业层面

房地产及相关行业就业人口：数千万人。

恒大倒下：
- 恒大员工失业
- 上游供应商裁员
- 下游装修、物业萎缩
- 建筑工人返乡

## 与2008年金融危机的类比

美国次贷危机的链条：
```
银行放贷给买房人
→ 打包成MBS
→ 评级机构背书
→ 卖给全球投资者
→ 违约率上升
→ MBS崩盘
→ 全球金融危机
```

中国房地产危机的链条：
```
购房者付款 + 银行贷款
→ 开发商拿钱
→ 多元扩张 + 分红 + 转移资产
→ 资金链断裂
→ 烂尾楼 + 供应商倒闭
→ 银行坏账 + 消费萎缩
→ 地方财政减收
```

**本质相同**：风险从一个人身上转移到整个社会。
