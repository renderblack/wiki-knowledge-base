# 猪肉周期研究

## 概述

猪肉是中国 CPI 的核心组成部分（约占 CPI 权重 2-3%），猪肉价格的周期性波动对通胀数据有显著影响。"猪周期"每轮约 3-4 年，由供给侧（能繁母猪存栏量）驱动。

## 周期机制

```
能繁母猪减少 → 10个月后生猪出栏下降 → 猪肉价格上涨
     ↑                                    ↓
能繁母猪补栏 ← 养殖利润丰厚 ← 产能扩张 ← 猪肉价格下跌
```

关键时间节点：
- 能繁母猪去化 → 传导到出栏约 **10个月**
- 疫病（非洲猪瘟等）会大幅放大周期振幅

## 核心指标

| 指标 | 说明 | 数据来源 |
|------|------|---------|
| 能繁母猪存栏量 | 核心先行指标，农业部每月发布 | 农业农村部 |
| 生猪价格 | 当期价格，月度频率 | 商务部、畜牧业协会 |
| 仔猪价格 | 补栏热情的前瞻信号 | 各地养殖场报价 |
| 猪粮比价 | 养殖利润临界点，6:1为盈亏平衡 | 计算得出 |
| 生猪出栏量 | 当期供给，月度/季度 | 国家统计局 |

## 2024-2025 年周期位置

> 状态：[[#⏳ 进行中 — 待补充数据]]

- 2024年中：猪价持续低迷，行业深度亏损，能繁母猪加速去化
- 2024年底-2025年初：猪价反弹，周期触底回升信号
- 2025年：关注产能回补节奏，决定新一轮高点时间

## 与宏观经济的关系

- **CPI影响**：猪肉权重约2-3%，但波动大时能解释CPI变动的30-50%
- **政策关注**：猪肉价格上涨过快可能触发储备投放（国家发改委）
- **与PPI联动**：养猪饲料（豆粕、玉米）与农产品/工业品价格互相传导
