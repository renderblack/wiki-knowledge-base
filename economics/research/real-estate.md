# 房地产市场研究

## 概述

房地产是中国经济最重要的部门，2021年峰值时房地产及相关产业链占GDP约25-30%。2021年下半年恒大暴雷后，行业进入漫长调整期，至今仍未触底。

## 核心矛盾

| 矛盾 | 说明 |
|------|------|
| 供给侧 | 开发商资产负债表受损，停工楼盘制约交付信心 |
| 需求侧 | 居民预期转弱，加杠杆意愿低，人口压力 |
| 价格压力 | 70城房价持续下行，一线城市出现分化 |
| 金融风险 | 土地财政萎缩影响地方政府收入（→ [[城投债]]压力） |

## 政策应对

### 已出台措施
- 取消限购限贷（多数城市）
- 降低首付比例、下调存量房贷利率
- 白名单贷款（央行支持合规项目）
- 收储去库存（政府收购商品房作保障房）

### 核心政策评估框架
1. **需求侧** — 是否降低购房成本、提升预期
2. **供给侧** — 是否改善开发商现金流
3. **金融侧** — 银行是否扩大信贷投放
4. **财政侧** — 地方收入是否有替代来源（→ [[化债]]）

## 房价下跌的宏观传导

```
房价下跌 → 财富效应 → 消费收缩
         ↓
居民房贷违约风险上升 → 银行资产负债表压力
         ↓
土地市场降温 → 地方卖地收入减少
         ↓
城投平台收入下降 → [[隐性债务风险]]上升
         ↓
→ 更大规模 [[化债研究|化债]]压力
```
