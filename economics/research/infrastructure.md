# 基建设施研究

## 概述

基础设施投资是中国逆周期调节的核心工具。经济下行期，政府通过增加基建投资托底需求；经济过热期，基建降温防止过热。与[[房地产市场]]和[[化债研究]]密切相关。

## 投资主体

| 主体 | 资金来源 | 特点 |
|------|---------|------|
| 铁路/公路 | 中央专项债 + 国铁集团 | 回报周期长，现金流稳定 |
| 市政/城投 | 城投债 + 土地出让金 | 与[[隐性债务]]高度相关 |
| 水利/能源 | 中央预算 + 专项债 | 政策导向强 |
| 新基建（数字） | 政府补贴 + 政策性银行 | 近年增速最快 |

## 核心指标

| 指标 | 频率 | 说明 |
|------|------|------|
| 基础设施投资完成额累计同比 | 月度 | 统计局口径，含电力/交通/水利等 |
| 城投债净融资规模 | 月度 | 城投债是基建重要资金来源 |
| 专项债发行规模 | 月度 | 地方债的重要组成部分 |
| 挖掘机/重卡销量 | 月度 | 基建强度的前置指标（"挖掘机指数"） |
| 螺纹钢表观消费量 | 周度 | 基建+地产的钢材需求代理指标 |

## 基建与城投债的关系

```
经济下行 → 财政发力 → 专项债放量 → 城投平台代建需求增加
                                  ↓
                          城投债净融资上升
                                  ↓
                          [[隐性债务]]规模扩大
                                  ↓
                          后续 [[化债研究|化债]]压力
```
